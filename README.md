# ff_danat

Telegram bot loyihasi. GitHub va serverga xavfsiz deploy qilish uchun maxfiy sozlamalar `.env` orqali boshqariladi.

## Setup

1. `requirements.txt` dagi paketlarni o'rnating.
2. `.env.example` dan nusxa olib `.env` yarating.
3. `.env` ichida `BOT_TOKEN`, `DATABASE_URL`, `SUPERADMIN_IDS` qiymatlarini to'ldiring.
4. Botni `python bot.py` bilan ishga tushiring.
